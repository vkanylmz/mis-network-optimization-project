BIST 100 Portfolio Diversification
via Minimum Spanning Tree (MST)
MIS Network Optimization Project  |  Solution Overview
1. Problem Definition
Portfolio managers investing in Borsa Istanbul (BIST 100) face a critical challenge: selecting stocks that do not move together, so that when one falls, others remain unaffected. This is known as portfolio diversification.
Holding multiple stocks from the same sector - for example, four different bank stocks - provides an illusion of diversification. In reality, when the banking sector drops, all four stocks fall simultaneously. The challenge is to uncover the hidden correlation structure between dozens of stocks and use it to make better investment decisions.
This project addresses that challenge by modeling stock relationships as a network and applying the Minimum Spanning Tree algorithm to extract the backbone structure of market correlations.

2.2  Edges
An edge between two stocks represents a measurable statistical relationship between their daily price movements. The complete graph contains 820 possible edges (all pairs). After applying the MST algorithm, only the 40 most structurally important edges remain.
2.3  Weights - Mantegna Distance
Edge weights are computed using the Mantegna (1999) distance metric, which converts Pearson correlation (rho) into a proper Euclidean distance:
d(i,j)  =  v( 2 × (1 - ???) )
A short edge (low distance) means high correlation - stocks move together. A long edge means low correlation - stocks move independently. This transformation allows the MST algorithm to treat correlation as a spatial distance problem.
3. Algorithm - Minimum Spanning Tree (Kruskal)
The Minimum Spanning Tree (MST) algorithm connects all nodes in the network using the minimum total edge weight, without creating any cycles. It acts as a filter - removing redundant connections and revealing only the strongest structural relationships in the market.
Kruskal's algorithm is used (via NetworkX):
• Sort all 820 edges by distance weight in ascending order
• Greedily add the shortest edge that does not create a cycle
• Stop when all 41 nodes are connected - result is 40 edges
Time complexity: O(E log E) where E is the number of edges.
The MST is the standard approach in financial econophysics for studying market correlation structure (Mantegna 1999, Onnela et al. 2003).

KCHOL (Koc Holding) and SAHOL (Sabanci Holding) emerged as the most connected nodes with degrees of 10 and 9 respectively.
