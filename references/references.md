# **Note on Data**
*Price data was retrieved using the yfinance library, which sources adjusted closing prices from Yahoo Finance. The analysis covers a 1-year period of daily returns. Log returns were used (as opposed to arithmetic returns) to ensure stationarity, following standard practice in quantitative finance.*

*The Mantegna distance metric used in this project was first proposed in: Mantegna, R. N. (1999) and has since become the standard approach for correlation-based network analysis in financial markets.*
